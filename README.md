# 🤖 Agentic AI Chatbot (LangGraph + Groq)

🚀 This project is an **Agentic AI Chatbot** built using:

* LangGraph (Agent workflow)
* Groq LLM
* Streamlit UI

👨‍💻 **Developed by: Kumar Basu Singh**

---

## 🔥 Features

* ✅ Basic Chatbot using LLM
* ✅ Modular Graph-based architecture
* ✅ Streamlit UI for interaction
* ✅ Groq API integration
* ✅ Extendable to tools & agents

---

## 🧠 Tech Stack

* Python
* LangChain / LangGraph
* Groq API
* Streamlit

---

## ⚙️ Installation

```bash
git clone https://github.com/mr-basu-singh/Agentic-ChatBot.git
cd Agentic-ChatBot
pip install -r requirements.txt
streamlit run app.py
```

---

## 🔑 Setup

Add your Groq API key in the UI or environment variable:

```bash
export GROQ_API_KEY=your_key_here
```
